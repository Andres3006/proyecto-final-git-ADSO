# ¿Qué es Git?

# 

# Git es un sistema de control de versiones que permite registrar los cambios de un proyecto a lo largo del tiempo. Ayuda a que varios desarrolladores trabajen en el mismo proyecto sin sobrescribir el trabajo de los demás. Con Git es posible crear ramas independientes para nuevas funciones y luego combinarlas con la rama principal. También permite volver a versiones anteriores del proyecto si ocurre algún error. Es una herramienta distribuida, lo que significa que cada usuario tiene una copia completa del historial. Git es muy usado en la industria porque facilita el trabajo colaborativo. Además, permite resolver conflictos cuando dos personas editan el mismo archivo. En resumen, Git es esencial para el desarrollo moderno de software.

# 10 comandos de Git explicados

# 

# git init – Crea un nuevo repositorio en la carpeta actual.

# 

# git add . – Agrega todos los archivos al área de preparación.

# 

# git commit -m "" – Guarda un cambio en el historial del repositorio.

# 

# git status – Muestra el estado actual de los archivos.

# 

# git branch – Lista todas las ramas del repositorio.

# 

# git checkout -b nombre – Crea una nueva rama y se mueve a ella.

# 

# git merge rama – Combina una rama con la rama actual.

# 

# git pull origin main – Descarga los cambios desde GitHub a tu PC.

# 

# git push origin rama – Sube tus cambios desde tu PC a GitHub.

# 

# git log – Muestra el historial completo de commits.
